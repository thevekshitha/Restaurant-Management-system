# 🍽️ Restaurant Management System

## 📌 Project Overview

This project is a **console-based Restaurant Management System** developed using Python.
It demonstrates the practical implementation of multiple **Data Structures** in a real-world scenario.

The system manages restaurant operations such as:

* Customer handling
* Order processing
* Billing system
* Menu management
* Statistics tracking

---

## 🚀 Features

* 📋 Menu displayed using **Tree structure**
* 👤 Customer management using **Hash Map**
* 🧾 Order history using **Linked List**
* 🍳 Kitchen orders using **Queue (FIFO)**
* 👑 VIP orders using **Priority Queue**
* 🧮 Billing system using **Stack**
* 📊 Popular items tracking using **Arrays**

---

## 🧠 Data Structures Used

| Data Structure | Purpose                      |
| -------------- | ---------------------------- |
| Linked List    | Store customer order history |
| Stack          | Manage recent bills          |
| Queue          | Process kitchen orders       |
| Priority Queue | Handle VIP customers         |
| Tree           | Organize menu categories     |
| Hash Map       | Store customer data          |
| Array          | Track statistics             |

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/thevekshitha/Restaurant-Management-system.git
```

2. Navigate to the folder:

```
cd Restaurant-Management-system
```

3. Run the program:

```
python restaurant.py
```
